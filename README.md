# Investopia - Investment Platform HTML Template

Investopia is a Bootstrap 5 based HTML template for an investment platform website and client dashboard flow.

## Demo

https://abuubaidah.github.io/investopia/

## Key Highlights

- Public website with pricing plans, calculator, about, contact, blog, and activity pages
- Dashboard-style inner pages for investments, deposits, withdrawals, referrals, KYC, profile, and support
- Plan detail flows for non-logged and logged users
- Submission success state page
- Unified navbar system and shared styling across all pages

2. Open `index.html` directly in a browser, or use any local static server.

- `assets/css/iv-styles.css` - main custom stylesheet
